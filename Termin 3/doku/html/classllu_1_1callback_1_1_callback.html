Detailed Description

template<typename C, typename D>
class llu::callback::Callback< C, D >

Calback manager class.

This class can be used for callbackmanagement, it impelemtnts functionality to use in a observer pattern the C is the context of a callback_registration usuali a void* D is the Data which must be given wen signaling listeners
