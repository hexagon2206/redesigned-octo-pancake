Detailed Description

template<typename E>
class llu::datastructs::LinkedListArray< E >

Implementation of a simple generik thread save Array.

internely this class is implemented as a linked List. therefor the differences in the indizes can be enormus, without much memory consumtion,

Note
compared to a regular array this is very slow, should not be used if a normal array would fit the task
